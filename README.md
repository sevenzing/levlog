### Currently avaliable on [levlog.ru](https://levlog.ru/)

## Run localy
1. Configure .env file

2. Run database

```
docker-compose up -d
```

3. Requirement

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

4. Run server

```
python manage.py runserser
```
